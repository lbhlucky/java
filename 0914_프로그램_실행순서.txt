< 프로그램 실행 과정 >
1. 소스 코드 작성 (*.java)

2. 소스 코드 번역하여 클래스 파일 생성(*.class)

3. 프로그램 실행( = 클래스 파일 실행)

4. 클래스 로딩 => static 키워드가 붙은 변수 및 메서드가 메모리에 로딩
		  (= 정적 멤버 = 클래스 멤버)

5. main() 메서드 호출(실행)

6. 인스턴스 생성(new 키워드) => 인스턴스 멤버 변수 및 메서드가 메모리에 로딩

7. 인스턴스 내의 메서드 호출 => 로컬변수가 메모리에 로딩

8. 결과 출력

9. 프로그램 종료
